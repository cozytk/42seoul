# minishell

**나만의 쉘 만들기**

### 구현되어 있는 기능
    - echo with '-n' option
    - cd
    - pwd
    - export
    - unset
    - env
    - exit
    - ;
    - ' and "
    - < > >>
    - |
    - $ENV
    - $?
    - ctrl-C, ctrl-D and ctrl-\
    - signal
